# 📚 Booko

Booko is a modern, community-driven marketplace for book lovers. Whether you're looking to find a rare edition, swap a read favorite, or sell your collection, Booko provides the perfect platform to connect with fellow bibliophiles.

![Booko Hero](src/assets/hero.png)

## 🌟 Features

- **Discover Books**: Browse a curated list of book advertisements from the community.
- **Post Listings**: easily create and manage ads for books you want to sell or swap.
- **Real-time Chat**: Connect directly with buyers and sellers through the integrated ChatHub.
- **Personalized Profiles**: Track your listings and manage your user information.
- **Secure Access**: Built-in authentication to ensure a safe community environment.

## 🚀 Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/booko.git
   cd booko
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
